IPv4 Calculator

About IPv4

IPv4 uses 32-bit addressing. CIDR prefix length and the subnet mask define the network and host portions of the address.

IPv4 uses a broadcast address, which is normally the final address in a subnet.

IPv6 Calculator

About IPv6

IPv6 uses 128-bit addressing. IPv6 does not use broadcast addresses; it uses multicast addresses for group communication.

Common IPv6 multicast groups include FF02::1 for all nodes and FF02::2 for all routers on the local link.

VLSM Auto-Allocation Engine

Enter one subnet per line. You can use just hosts, or name = hosts.

This mode splits the base network into equal-size subnets. Example: 192.168.10.0/24 into 4 networks creates four /26 subnets.

About VLSM

Variable Length Subnet Masking allocates different subnet sizes from the same parent network.

Best practice: allocate the largest requirements first. This reduces fragmentation and improves address-space efficiency.
